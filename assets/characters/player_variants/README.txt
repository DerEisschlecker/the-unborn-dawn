Player appearance templates for the character creator.

The PNG previews in this folder use selected CC0 character poses from Kenney's free character packs. Only the small files needed by the current creator UI are included here, keeping the project light while still using real free templates.

Mapping:
female_wanderer.png - Kenney Platformer Characters / Female / stand
male_wanderer.png - Kenney Platformer Characters / Player / stand
female_mechanic.png - Kenney Toon Characters / Female person / idle
male_mechanic.png - Kenney Toon Characters / Male person / idle
female_medic.png - Kenney Toon Characters / Female adventurer / hold
male_medic.png - Kenney Toon Characters / Male adventurer / hold
female_guardian.png - Kenney Platformer Characters / Soldier / stand
male_guardian.png - Kenney Platformer Characters / Soldier / stand
